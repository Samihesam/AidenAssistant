# دستیار شخصی ایدن (Aiden Personal Assistant)

یه دستیار هوشمند شخصی متن‌باز برای اندروید — با پشتیبانی از پیامک/تماس، گوش‌به‌زنگی همیشگی، صداهای متنوع، آواتارهای قابل‌انتخاب، چندزبانگی، رادار افراد نزدیک، و شخصی‌سازی کامل با فرمان صوتی.

## ✨ ویژگی‌ها
- خوندن/ارسال پیامک و مدیریت تماس (با اجازه‌ی صریح کاربر)
- حالت «همیشه به گوش باش» با کلمه‌ی بیدارباش
- ۱۰ صدای مختلف (۵ زن + ۵ مرد)
- چند آواتار قابل‌انتخاب: جادوگر، ربات، صورت زرد ناز، و بیشتر...
- پشتیبانی از ۱۱ زبان
- رادار پیدا کردن افراد نزدیک و چت زنده (۱۸+، با رعایت اصول ایمنی)
- **شخصی‌سازی کامل با فرمان صوتی**: تغییر اسم دستیار، به‌خاطر سپردن ترجیحات کاربر

## 📂 ساختار پروژه
```
/app          → کد اپلیکیشن اندروید (Kotlin + Jetpack Compose)
/backend      → سرور رادار و چت (Node.js + Express + Socket.io)
LICENSE       → مجوز MIT
```

## 🚀 راه‌اندازی
۱. مخزن رو کلون کن
۲. پوشه‌ی `app` رو توی Android Studio باز کن (نیازمند Kotlin + Jetpack Compose)
۳. برای بخش رادار، پوشه‌ی `backend` رو طبق راهنمای داخلش راه‌اندازی کن

## ⚠️ نکات مهم قبل از استفاده
- برای انتشار عمومی روی گوگل پلی، باید Privacy Policy واقعی بنویسی (به‌خصوص برای دسترسی‌های حساس مثل پیامک، تماس، Accessibility Service)
